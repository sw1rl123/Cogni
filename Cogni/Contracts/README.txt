	В этой папке будут создержаться контракты для общения с фронтенд частью.
	Контракты - это "соглашения" о том в каком виде будут переданы данные. 
Например, с фронтенда пришел запрос на создание пользователя. 
Данные, которые пользователь ввел в форме на странице, упаковываются в JSON и отправляютсв теле HTTP Post запроса. На сервере они автоматически распаковываются в экземпляр класса, указанный для данного метода контроллера.

 public record SignUpRequest //пример контракта для данных регистрации
    (
        string Name,
        string Login,
        string password
    );
